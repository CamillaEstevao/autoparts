# AutoParts Prime

Projeto React + Vite para vitrine premium de autopeças.

## Rodar no computador

```bash
npm install
npm run dev
```

Depois abra o link que aparecer no terminal, normalmente:

```txt
http://localhost:5173
```

## Onde trocar WhatsApp

No arquivo `src/App.jsx`, altere:

```js
const whatsappNumber = "5511999999999";
```

Use DDI + DDD + número, sem espaços.
